## SALAMI Data Download Script

To download the music files used in the dataset, you can use the `SALAMI_download.py` script located in the `src` directory. This script downloads all song files from the provided URLs in the `id_index_internetarchive.csv` file.

### Usage

To use the script, navigate to the `scripts` directory and run:

```bash
python SALAMI_download.py <path_to_csv_file> <output_directory>
